### This website was made with little to no help from AI, genuinely made by my hands.

# Home
### This section Features an animation effect of someone writing with sound playing in the background when its the first time loading into the website. The writing animation effect was achieved by ***svg text*** attributes whilst the sound playing ability was from js event listener to play sound every time the website was refreshed. When entering you will see the animations of top bar, bottom buttons popping up like pressed buttons being set free when someone comes to seek them.

# About
### This section features simple animations. In the image, the animations was simple and achieved by ***scaleY*** and ***transistion origin: center***. it is a simple section just featuring some things about me.

# Skills
### This section containers one hollow letter saying ***SKILLS*** which is animated with the help of ***svg text*** attributes used from the initial page's middle header animatiion keyframes, Other elements like ***MY TOOLBOX*** is animated by ***fadeIn*** keyframes which is widely used in this site. The box with texts "Front End" and "Back end" is animated by ***translateY*** simply. and features little buttons inside the box which leads you to the respective wiki.

# Projects
### This projects section features my works that I feel best show my knowledge. The starting left side text was animated the same as that of *1st* and *3rd* section's method. The projects card feature : image container, header, paragraph and specially the live and code button. The live and code buttons leads you to the live code and leads you to the github where the code is respectively. It was initially animated by what I call "Stretch animation effect" where the card's width goes form 1 to 900 so it looks good but later on as I finished up the final section I changed the card animation ***animateCard*** keyframes in css to just prop up when in view from JS *InteractionManager*.

# Contact
### This section of the web features a contact page where people who are interested to work contact me via email as a formality. all the card in this section is animated. The red card is animated with a little "heartbeat" animation effect from css which you can find in the last parts. The interesting part is of submit button. When the form is empty the submit button just clicks but when its filled it plays a little animation to let you know that your mail is sent to me.

## Dark theme
### When an user presses the little window icon in the navigation bar then it plays a little animation to let you know its on. All the elements having dark theme have class called ***windowDown*** which is then added when clicked on the window icon with the help of js.

## Local time theme management
### This website features (2 theme) based on local time of user set in the browser. When the clock hits 6pm the website automatically changes into dark theme to better help user's eye to lessen eye strain so the user have it easy when viewing the website.
